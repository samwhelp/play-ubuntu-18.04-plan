
# Prototype play-pekwm

## Usage


### install

``` sh
$ ./install.sh
```

or

``` sh
$ make install
```


### package-install

``` sh
$ ./package-install.sh
```

or

``` sh
$ make package-install
```


### config-install

``` sh
$ ./config-install.sh
```

or

``` sh
$ make config-install
```


## Keyboard shortcuts

Please refer to

* $ less ~/.pekwm/keys
